# Session Start

Initialize a new development session with current context.

## Do the following:

1. **Check current date** and create/identify Testing folder (format: `YY-MM-DDx`)

2. **Show current stable agent version** from `retell/agents/`

3. **Check for uncommitted changes** in both retell and n8n repos

4. **Quick health check:**
   - List agents in current Testing folder
   - Show any CHANGELOG.md entries from today

5. **Report session context:**
```
═══════════════════════════════════════════
SESSION START
═══════════════════════════════════════════
Date: [date]
Working folder: retell/Testing/[folder]/
Stable version: vX.XX
Latest dev version: vX.XX (if different)
Uncommitted changes: [yes/no]
═══════════════════════════════════════════
```

## Then ask:
"What would you like to work on today?"
